DESCRIPTION:
------------

Alexz – Creative Agency & Personal One Page/Multipurpose HTML Template. The template is designed for any type of Design Agencies, Portfolios, Corporate websites, and others. The template is Super Easy To Use. Based on custom Opacity CSS, and it’s also easy to customize. It has 4 HTML files. Everything was built with Clean and Easy to Understand Coding using custom CSS, No framework used here. The Template Design Is Flat And Simple. This Template Gives Better Browsing And Great Experience.
